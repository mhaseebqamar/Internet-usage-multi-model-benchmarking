# Predictive Analysis of Internet User Demographics

📌 Overview

This project performs a comprehensive comparative analysis of 5 different machine learning models to predict user segments based on demographic data. Utilizing a dataset of 10,000+ records, the study explores the relationship between early digital adoption and user characteristics like education, income, and age.

🛠️ Technical Stack

Language: Python

Models: SVM, Decision Tree, Logistic Regression, Neural Networks (MLP), and K-Nearest Neighbors (KNN).

Libraries: Scikit-Learn, Pandas, NumPy, Matplotlib, Seaborn.

Techniques: Data Preprocessing, Label Encoding, Feature Scaling, Cross-Validation, and ROC/AUC Evaluation.

🚀 Key Results

Top Performer: SVM achieved the highest classification accuracy (66.93%).

Best Interpretability: Logistic Regression yielded the highest AUC (0.757), providing the most reliable insights for behavioral prediction.

Visualization: Generated a comprehensive comparison of all models using Heatmaps and ROC Curves to visualize the trade-offs between precision and recall.

📈 Business & Social Insights

Determined that demographic features provide moderate predictive power for user classification.

Demonstrated that classical ML models can outperform complex Neural Networks on structured demographic datasets.

Provided a framework for understanding the "Digital Divide" and technology diffusion patterns.
